## 📌 Задача: создание sparse-файлов

Реализовать программу на C для создания **разреженных (sparse) файлов**.

### Требования

* Использовать системные вызовы:
  `open`, `read`, `write`, `lseek`, `ftruncate`
* Обрабатывать данные **поблочно**

  * размер блока по умолчанию: `4096`
  * поддержка параметра `-b <size>` через `getopt`
* Если блок состоит только из нулей → использовать `lseek`, иначе `write`

### Режимы работы

* `./myprogram output` → чтение из `stdin`
* `./myprogram input output` → копирование файла

### Тестирование

1. Скрипт создаёт файл `A`:

   * размер: `4MB + 1 байт`
   * почти весь из нулей
   * байты `1` по смещениям: `0`, `10000`, `конец`

2. В `runme.sh`:

   * создать `A`
   * сделать sparse-копию `B`
   * сжать `A` и `B` через `gzip`
   * восстановить `B.gz` → `C` через программу
   * создать `D` с блоком `100`
   * вывести размеры через `stat`

### 📊 Проверка

Сравниваются:

* логический размер файлов
* реально занятое место (sparse-файлы занимают меньше)
